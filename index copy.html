注力作品：タートルアドベンチャー

Unity C# Nintendo SDK Nintendo Switch CharacterController 状態遷移管理 3Dアクション開発 UI/UX設計 生成AI活用開発

ゲーム概要：片手操作で楽しめる、スマートフォン向けの本格3Dアクションゲームです。戦闘・ギミック・探索要素を組み合わせ、没入感のあるプレイ体験を目指しています。現在(2025/7～)はNintendo Switch向けへの移植にも取り組んでおり、マルチプラットフォームを見据えた設計や、入力デバイス・UXの違いに対応した改善を進めています。

開発目的：敵AI、状態遷移、ギミック演出、ステージ構成など、幅広い技術の習得と応用力の強化を目的に開発しています。

開発期間：200時間以上(2023年5月より他ゲームと並行して開発中)

OrderGuessingPuzzle - 順番当てパズル（2025年7月リリース）

Unity C# Google AdMob 生成AI活用開発 ランダム生成 UIアニメーション パズルロジック設計

ゲーム概要：ランダムに配置されたオブジェクトをヒントを頼りに順番通りに並び替えるマスターマインド風ゲームです

成果：生成AIを活用することでステージ生成ロジックやアニメーションを効率よく実装し、短期間でプロトタイプ開発→リリースに成功しました。

開発期間：約30時間（2025年6月～2025年7月）

デスクトップマネージャー（2025年7月制作）

Unity C# UI設計 時間管理ゲーム設計 タスク設計 ドラッグ＆ドロップ

ゲーム概要：段取り力が試されるゲームです。制限時間内に注文通りのフォルダを作成して提出して高スコアを狙うことが目標です。
フォルダの圧縮やポップアップによる進行阻害、アイテムのマージなど複数のタスクを段取り良くこなす力が試されます。

開発目的：カジュアルゲームだけでなく、一風変わったゲームを作成したいという想いから始めたプロジェクトです。

開発期間：約10時間（2025年7月15～現在）

憶えてすすめ！- 記憶×迷路パズル（2025年3月リリース）

Unity C# Google AdMob JSON 外部ツール作成 データ駆動設計 マップ生成

ゲーム概要：ゴールへの道が徐々に消えていく迷路パズルです。プレイヤーは記憶力を頼りに、ギミックを乗り越えながら進んでいくユニークな設計となっています。

開発目的：ステージ運用の効率化を学ぶことを目的に開発しました。

成果：JSONファイルを活用したマップ生成と、視覚的に編集可能な外部ステージエディタを開発し、制作効率を大幅に向上させました。

リリース実績：GooglePlay

開発期間：約70時間（2024年12月～2025年3月）

GooglePlay

スライドパズル - エンドレス（2025年7月リリース）

Unity C# Google AdMob グリッド管理 難易度設計 ランダム生成

ゲーム概要：シャッフルされたパネルを指定された順番に並び替えるスライドパズルです。ステージが上がるごとにグリッドやパネルの種類を増やし難易度を向上させ、飽きさせない工夫をしました。

開発目的：簡易的な思い付きを短期間で形にする練習を目的としました

開発期間：約20時間（2025年7月1～2025年7月10）

使用技術：Unity, C#, Google AdMob, グリッド管理, 難易度設計, ランダム生成

Unity C# Google AdMob ターン制ゲーム設計 状態管理 UI設計

開発目的：短期間での開発を目標に制作。スケジュール管理と迅速な実装力を養うことを目的としました。

TwinShips - 隕石を避けろ！（2022年リリース）

Unity C# Google AdMob 同時操作制御 入力管理 アクションゲーム設計

ゲーム概要：飛来する隕石を左右2つの宇宙船で回避し続ける、反射神経と集中力が試されるアクションゲームです。左右の同時操作が求められることで、脳トレ効果も期待できます。

開発目的：初めて発信するアプリという事もあり、ユーザーからのフィードバックを得ることを主な目的とし、シンプルな操作性とテンポの良さを意識して開発しました。

成果：実際のユーザーから得た意見をもとに、UIや難易度の調整を行う経験ができました。また、自分の作品に対して直接ユーザーの声を聞くことが出来る事にやりがいを実感しました。

リリース実績：GooglePlay

開発期間：2022年1月いっぱい

デモプレイ(PC) GooglePlay
